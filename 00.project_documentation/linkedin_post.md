🚀 Sharing a new analytics engineering project!

An end-to-end data warehouse with data visualization and data analysis I built for a fictional manufacturing company (LOMPSTAR), replacing a manual Excel-based process for tracking manufacturing KPIs.

Full workflow: standardized data entry (Excel + VBA) → Extract & Load (Python) → warehousing (SQL Server or Azure SQL Server) → dashboard (Power BI).

A few techniques and tools used on this project:
    🔹VBA for data standardization from Excel;
    🔹Python EL pipeline (pandas, pyodbc & openpyxl) to extract data from excel files and load it into a Database;
    🔹Data Warehouse with Medallion Architecture (Bronze, Silver, Gold) using T-SQL window functions, indexes and CTEs;
    🔹EDA in Jupyter Notebook (SQLAlchemy & pandas) to identify outliers and wrong data patterns;
    🔹Three Power BI reports with customized HTML and CSS visuals.

⚠️ All data used was generated by AI - The company, products and people are invented. All the measures and KPIs are based on personal experience and public sources, properly referenced in the project's documentation.

I also want to thank Baraa Khatib Salkini, whose SQL Data Warehouse content shaped the medallion structure in this project and many more I've built in real life. If it wasn't for his free content, I would never have started working in this field, or wanted to learn more about data. Really glad people like him exist! 🙏

If you are interested in using HTML and CSS for custom visuals in low-volume Power BI datasets, I truly recommend Evelane Santos - that's where I got the visuals I edited and customized!
