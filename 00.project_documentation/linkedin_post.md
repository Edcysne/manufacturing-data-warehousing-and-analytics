🚀 Sharing a new analytics engineering project!

An end-to-end data warehouse I built for a fictional manufacturing company
(LOMPSTAR), replacing a manual Excel-based process for tracking production
OEE (Overall Equipment Effectiveness).

Full workflow: standardized data entry (Excel + VBA) → Extract & Load
(Python) → warehousing (SQL Server) → dashboard (Power BI).

A few techniques and tools used on this project:
- VBA for data standardization from Excel;
- Python EL pipeline (pandas, pyodbc & openpyxl) to extract data from excel files and load on a Database;
- Medallion architecture (Bronze / Silver / Gold) using T-SQL Window functions, indexes and CTEs;
- EDA in Jupyter Notebook (sqlalchemy & pandas) to identify outliers and wrong data patterns;
- Three Power BI reports with customized HTML and CSS visuals.

All data used was generated by AI - The company, products and people are invented. All the measures and KPIs
are based on personal experience and public websites, properly referenced on the projects documentation.
